# Dotfiles
## Installation
Install XCode.

Run fallowing command in home directory.
```
git clone git@github.com:widmogrod/Dotfiles.git
cd ~/Dotfiles
./bin/bootstrap
./bin/build-vim
./bin/development
```

## Refereces

```
mk - move line up
mj - move line donw
<leader>/ - toggle inline comment
<leader>? - toggle block comment
```

```
cs"' - change braces
cd'<p> - ....
```

```
<leader>q - kill buffer
<leader>Q - force kill buffer
```

### Context specific

```
<leader>i - inspector
```

### Html
```
<c-y>, expand emmet
```
