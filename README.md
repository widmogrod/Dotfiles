# Dotfiles
## Installation

```
git clone git@github.com:widmogrod/Dotfiles.git && cd ~/Dotfiles && ./build
``

```
cd ~/Dotfiles/bash && ./vimproc.sh && ./vim-install-ycm.sh && ./install-composer.sh && ./install-php-libs.sh
```

## Shortcuts

mk - move line up
mj - move line donw
\/ - toggle inline comment
\? - toggle block comment
